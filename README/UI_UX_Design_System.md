# 视觉与设计系统 (UI/UX Design System)

## 1. 视觉灵魂：星光感 (Starlight Aesthetic)
- **背景**: 深蓝色渐变至深紫色 (`#0f172a` to `#1e1b4b`)，带有微弱的 Canvas 粒子闪烁效果。
- **磨砂玻璃**: 组件使用 `backdrop-blur-md`，半透明边框，模拟星光下的质感。
- **色彩板**: 
    - 主色 (文字): `Slate-100` (近纯白)
    - 辅助色 (星光): `Amber-200` (淡淡的暖金)
    - 警示色: `Rose-400`

## 2. 核心交互
- **翻书动画 (Page Flip)**: 
    - 使用 `framer-motion`。
    - 水平轴 3D 旋转：`rotateY: -180deg`，模拟纸张翻动感。
- **卡片布局**: 
    - 采用 **Masonry (瀑布流)** 布局。
    - 游客点击点赞或评论：页面进行平滑的 `opacity: 0` 淡出，并重定向至 `/login`。

## 3. AI 交互体验 (Debounced Suggestion)
- 用户输入时，输入框右下角显示微弱的星光呼吸灯图标。
- 停止输入后，AI 润色文字以“星尘聚拢”的动画淡入显示在预览区，用户可一键替换。

## 4. 响应式规则
- **Mobile (优先)**: 全宽卡片，底部导航栏。
- **PC**: 最大容器宽度 `max-w-4xl`，居中显示，左右留白以增强沉浸感。

---

### 1. 视觉诊断：现在的 UI 差在哪？

* **背景太硬：** 纯白色背景在深夜非常刺眼，且没有空间深度。
* **构图发散：** 文字居中虽然不错，但由于周围没有任何装饰或容器，显得很单薄。
* **缺乏品牌感：** 字体和按钮都是浏览器默认风格，没有“星光”的精致感。

---

### 2. UI 升级方案：打造“星光沉浸感”

我建议在你的 `UI_UX_Design_System.md` 中增加以下具体的 CSS 指令，让 Windsurf 帮你重写样式：

#### A. 引入“深夜星空”渐变背景

不要用纯黑，要用深蓝色到深紫色的微弱渐变，并添加噪点（Noise）或微粒子效果。

```css
/* 参考色值 */
background: radial-gradient(circle at center, #1e1b4b 0%, #0f172a 100%);

```

#### B. 增加“毛玻璃”卡片容器

不要让文字直接飘在背景上，给它一个半透明的容器，增加层次感。

* **材质：** `backdrop-blur-xl` + `bg-white/5`。
* **边框：** 极细的白色半透明边框，模拟月光勾勒的边缘。

#### C. 字体升级

* **句子：** 使用中文字体中的**宋体/楷体变体**（如“思源宋体”），并加大字间距。
* **按钮：** “换一个”按钮改为药丸形状，带有一点点暖金色的发光（Glow）效果。

> 4. **按钮**：‘换一个’按钮改为半透明暖金色，悬停时有微弱的发光效果。
> 5. **右上角图标**：请使用 Lucide-react 图标库，将图标改为带磨砂质感的圆形按钮。”

### 4. 交互补全：那个“翻书”动画

现在的截屏是静态的。当你点击“换一个”时，如果没有动画，会显得很生硬。

* **建议：** 让 Windsurf 引入 `framer-motion`。
* **效果：** 点击时，旧句子向左 3D 旋转消失，新句子从右侧“翻”出来。


## 5. 个人主页规范 (Profile & Dashboard)
- **氛围**: 延续“星空”主题，但背景色可稍淡，以增加“安静回溯”的感觉。
- **功能组件**:
  - **情绪星座图**: 展示用户最近 30 天的情绪波动。
  - **管理抽屉**: 点击自己的帖子时，从右侧滑出半透明毛玻璃抽屉，用于编辑或删除。
  - **导出按钮**: 位于页面右上角，采用渐变边框按钮，点击后触发粒子效果。
